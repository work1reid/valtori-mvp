## 🔷 GitHub Pages

### Prerequisites
- GitHub account
- Repository created
- Code pushed to GitHub

### Deployment Steps

**1. Push Your Code**
```bash
# Navigate to your project
cd ~/Desktop/valtori-mvp-deploy

# Initialize git (if not already done)
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit - Valtori AI MVP"

# Add remote (replace with your repo URL)
git remote add origin https://github.com/YOUR-USERNAME/valtori-ai.git

# Push to GitHub
git push -u origin main
```

**2. Enable GitHub Pages**
1. Go to your repository on GitHub
2. Click **Settings** tab
3. Scroll to **Pages** section (left sidebar)
4. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**

**3. Wait for Deployment**
- Usually takes 2-5 minutes
- You'll see: "Your site is published at..."
- Visit: `https://YOUR-USERNAME.github.io/valtori-ai/`

**4. Set index.html as Homepage**

If using our file structure, rename `valtori-sales-simulator.html` to `index.html`:

```bash
mv valtori-sales-simulator.html index.html
git add .
git commit -m "Rename to index.html for GitHub Pages"
git push
```

### Troubleshooting

**Issue**: 404 Error
- **Fix**: Make sure file is named `index.html`
- **Fix**: Check branch is `main` in settings
- **Fix**: Clear browser cache

**Issue**: Updates not showing
- **Fix**: Clear cache (Cmd+Shift+R on Mac)
- **Fix**: Wait 5 minutes for GitHub to rebuild
- **Fix**: Check GitHub Actions for build errors

## 🟣 Netlify

### Method 1: Drag & Drop (Easiest)

**1. Prepare Files**
```bash
# Create a deployment folder
mkdir deploy
cp index.html deploy/
```

**2. Deploy**
1. Go to [netlify.com](https://netlify.com)
2. Sign up / Log in
3. Drag the `deploy` folder onto the page
4. Done! You'll get a URL like: `random-name-123.netlify.app`

### Method 2: GitHub Integration

**1. Connect Repository**
1. Go to netlify.com
2. Click "New site from Git"
3. Choose GitHub
4. Select your repository
5. Configure:
   - Branch: `main`
   - Build command: (leave empty)
   - Publish directory: `/`
6. Click "Deploy site"

**2. Automatic Deploys**
- Every push to `main` triggers redeploy
- Get preview URLs for PRs
- Rollback to previous versions easily

### Custom Domain on Netlify

1. Go to Site Settings → Domain management
2. Click "Add custom domain"
3. Enter your domain (e.g., `valtori.ai`)
4. Follow DNS configuration instructions
5. SSL certificate auto-generated (HTTPS enabled)

## ▲ Vercel

### Method 1: CLI Deployment

**1. Install Vercel CLI**
```bash
npm install -g vercel
```

**2. Deploy**
```bash
cd ~/Desktop/valtori-ai
vercel

# Follow prompts:
# - Set up and deploy? Y
# - Which scope? (choose your account)
# - Link to existing project? N
# - Project name: valtori-ai
# - Directory: ./
# - Override settings? N
```

**3. Production Deploy**
```bash
vercel --prod
```

### Method 2: GitHub Integration

**1. Import Project**
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import from GitHub
4. Select repository
5. Keep default settings
6. Click "Deploy"

**2. Automatic Deploys**
- Main branch → Production
- Other branches → Preview URLs
- Pull requests → Automatic previews

### Environment Variables (For Future API Keys)

1. Go to Project Settings
2. Navigate to Environment Variables
3. Add:
   - `ANTHROPIC_API_KEY` (when moving to server-side)
   - Other secrets as needed

## 🌐 Custom Domain Setup

### Prerequisites
- Own a domain (e.g., valtori.ai)
- Access to DNS settings

### Option 1: Netlify Custom Domain

**1. In Netlify Dashboard:**
1. Site Settings → Domain management
2. Add custom domain: `valtori.ai`
3. Add `www.valtori.ai` as alias

**2. DNS Configuration:**

Add these records to your domain provider:

```
Type    Name    Value
A       @       75.2.60.5
CNAME   www     your-site.netlify.app
```

**3. SSL Certificate:**
- Auto-generated by Netlify
- HTTPS enabled automatically
- Takes 24-48 hours to propagate

### Option 2: GitHub Pages Custom Domain

**1. In Repository Settings:**
1. Settings → Pages
2. Custom domain: `valtori.ai`
3. Save

**2. DNS Configuration:**

```
Type    Name    Value
A       @       185.199.108.153
A       @       185.199.109.153
A       @       185.199.110.153
A       @       185.199.111.153
CNAME   www     YOUR-USERNAME.github.io
```

**3. Create CNAME file:**

```bash
echo "valtori.ai" > CNAME
git add CNAME
git commit -m "Add custom domain"
git push
```

## 🔄 Continuous Deployment

### GitHub Actions (Optional)

Create `.github/workflows/deploy.yml`:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      
      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./
```

This auto-deploys on every push to main.

## 🆘 Common Issues

### Issue: Site won't load
**Solutions:**
- Check browser console for errors
- Verify index.html exists
- Clear browser cache
- Check deployment logs

### Issue: Voice doesn't work
**Solutions:**
- Verify HTTPS (required for mic access)
- Check browser permissions
- Test in Chrome/Edge only
- Confirm microphone connected

### Issue: AI not responding
**Solutions:**
- Check internet connection
- Verify Claude API is accessible
- Check browser console for API errors
- Try different browser
